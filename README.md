## ⚡ Why Zyruz Booster?

Most "optimizing" tools do nothing but clear temporary files. **Zyruz Booster is different.** We go deep into the Windows Kernel, Registry, and Network Stack to eliminate the bottlenecks that are holding you back.

> [!IMPORTANT]
> Zyruz Booster is a **portable** application. No installation required. Just run the `.exe` as Administrator and dominate.

## 🔥 Key Features

### 🎮 FPS Boost & Performance
Stop the stuttering and maximize your frames. Zyruz modifies system-level scheduling and power states to ensure your hardware is always at peak performance.
- **Ultimate Power Plan:** Unlocks maximum CPU & GPU power states.
- **CPU Core Unparking:** Prevents mid-frame stutters by keeping all cores active.
- **HAGS Optimization:** Reduces GPU latency via Hardware Accelerated GPU Scheduling.
- **Kernel Locking:** Keeps the Windows kernel in RAM to avoid slow disk paging.

### 🖱️ Input Lag Reduction
In competitive gaming, every millisecond counts. Zyruz optimizes your HID stack for the most responsive feel possible.
- **Raw 1:1 Input:** Disables all forms of mouse acceleration and smoothing.
- **FSE Optimization:** Forces true exclusive fullscreen for zero display latency.
- **USB Priority:** Prevents USB sleep cycles and elevates HID IRQ priority.

### 🌐 Network & Ping Optimizer
Eliminate lag spikes and get the lowest ping to game servers.
- **Nagle's Algorithm:** Disabled packets buffering for instant data transmission.
- **Auto-DNS Optimizer:** Detects and applies the fastest DNS servers (Cloudflare/Google).
- **TCP Auto-Tuning:** Stabilizes packet flow to prevent variance spikes.

### 🧹 System Maintenance
Keep your system lean and mean.
- **RAM Cleaner:** Intelligent memory management that frees up resources without crashing apps.
- **Windows Debloater:** Removes unnecessary telemetry and background services.
- **Privacy Shield:** Stops Windows from phoning home and eating bandwidth.
